We take security seriously because we understand that we're handling your core data and workflows. Your business processes, sensitive information, and operational data are entrusted to us, and we treat this responsibility with the highest level of care.

## Data Handling

## Infrastructure Security

- **API key management**: Keys are cryptographically generated and can be rotated or revoked instantly from your dashboard.
- **Rate limiting**: Per-organization rate limits protect against abuse.
- **Automatic revocation**: Keys are automatically disabled if billing thresholds are exceeded.
- **Audit logging**: All API key creation, deletion, and usage events are logged.

## Enterprise Security

For organizations with additional security requirements, we offer:

- **Dedicated endpoints** with isolated compute and no shared infrastructure. See [Dedicated Endpoints](/enterprise/dedicated).
- **On-prem deployment** to run inference in your own cloud or data center. See [On-Prem](/enterprise/on-prem).
- **Custom security reviews** where we work directly with your security team.
