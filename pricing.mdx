## How Pricing Works

We offer five agent engines with different pricing tiers based on your needs:

### Available Engines

| Engine | Type | Input Tokens | Output Tokens |
|--------|------|--------------|---------------|
| **TIM** | Unified | $2.00 per 1M tokens | $8.00 per 1M tokens |
| **TIM-Edge** | Unified | $0.50 per 1M tokens | $2.00 per 1M tokens |
| **TIM-CLAUDE** | Compound | $4.00 per 1M tokens | $15.00 per 1M tokens |
| **TIM-CLAUDE-Heavy** | Compound | $6.00 per 1M tokens | $25.00 per 1M tokens |

### Input vs Output Tokens

**Input Tokens** include:
- Your initial prompt
- Tool use response values injected into to the context window during an agent run.

**Output Tokens** include:
- Only what the model generates. This includes reasoning tokens, parameters generation, and the final result.
